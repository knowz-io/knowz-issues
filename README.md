# Knowz Public Issue Tracker

[Knowz](https://knowz.io) is an AI-powered knowledge management platform that helps teams capture, organize, and retrieve institutional knowledge.

This repository is the **public issue tracker** for Knowz. Use it to:

- **Report bugs** you've encountered
- **Request features** you'd like to see
- **Ask questions** in [Discussions](https://github.com/knowz-io/knowz-issues/discussions)
- **Share ideas** and feedback with the community

## How to contribute

### Found a bug?

[Open a bug report](https://github.com/knowz-io/knowz-issues/issues/new?template=bug_report.yml) with steps to reproduce.

### Have a feature idea?

[Open a feature request](https://github.com/knowz-io/knowz-issues/issues/new?template=feature_request.yml) describing the problem and your proposed solution.

### Have a question?

Head to [Discussions Q&A](https://github.com/knowz-io/knowz-issues/discussions/categories/q-a) â€” the community and our team are happy to help.

### Found a security vulnerability?

Please **do not** open a public issue. See [SECURITY.md](docs/SECURITY.md) for responsible disclosure instructions.

## Issue lifecycle

1. You file an issue
2. Our team triages it and may ask follow-up questions
3. Once confirmed, it's tracked internally for implementation
4. When resolved, the issue is closed with a note about the fix
